Six Degrees Of Spotify

Overview: Six Degrees of Spotify is a web app that finds the shortest path between 2 popular Spotify artists using data collected from Spotify's 'fans also listen to' feature and a breadth first search (BFS) algorithm. The app was built using React.js, Node.js, and Express.js.

Features: Full-stack development, breadth first search (BFS), RESTful API development, 100% API test coverage with Jest

Features: User authentication, global search, document creation and deletion, database protection rules

Assignments, but Automated

Features: HTTP GET and POST requests, task automation, REST API calls, open source project

AVL Tree Container

Overview: Implemented a self-balancing AVL Tree structure to store users' names and id numbers in an optimal manner. The structure allows for inserting, searching, deleting, and traversing user information. Additionally, 11 unit tests were conducted to ensure each tree function works as intended.

Features: Optimized logarithmic lookup times, unit testing with Catch2, user-friendly APIs, object oriented programming

Features: Custom iterators, simulation testing and comparison, user-friendly APIs, object oriented programming
