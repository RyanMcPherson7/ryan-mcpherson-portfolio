Projects

Six Degrees Of Spotify

Overview: Developed a full-stack web app that finds the shortest path between 2 Spotify artists using data from Spotify's "fans also listen to" feature. The frontend was build with React.js and the backend with Node.js and Express.js. The backed graph contains ~10,000 vertices and ~100,000 edges.

Features: Full-stack development, breadth first search (BFS), RESTful API development

Assignments, but Automated

Overview: Built a script that automatically uploads homework assignments from my Canvas LMS account to a Notion database for quick access. The database displays the assignment name, due date, and parent course. Data is retrieved and posted using both services' REST APIs.

Features: HTTP GET and POST requests, task automation, REST API calls

AVL Tree Container

Overview: Implemented a self-balancing AVL Tree structure to store users' names and id numbers in an optimal manner. The structure allows for inserting, searching, deleting, and traversing user information. Additionally, 11 unit tests were conducted to ensure each tree function works as intended.

Features: Optimized logarithmic lookup times, unit testing with Catch2, user-friendly API, object oriented programming (OOP)

Hash Map Comparator

Overview: Handcrafted both an ordered hash map (AVL Tree backed) and an unordered hash map (separate chaining) to compare their performances. Additionally, each map's performance was also tested against the hash maps defined by C++'s Standard Library (STL).

Features: Custom iterators, simulation testing and comparison, user-friendly APIs, OOP

Minesweeper Clone

Overview: Recreated the popular retro game Minesweeper using the SFML C++ library to display a GUI for users to interact with. The game includes standard game features like flagging mines, displaying score, and win/loss conditions.

Features: Recursive tile reveal algorithm, user-friendly GUI, additional developer buttons, OOP
